# LOGIN AND REGISTRATION SYSTEM

## PROJECT OVERVIEW

This project is a simple console-based Login and Registration System developed in C++ as part of the CodeAlpha C++ Programming Internship.

The main purpose of this project is to understand how user authentication works using file handling in C++. The system allows users to create a new account and log in using their registered username and password.

All user data is stored in text files, making the project simple, lightweight, and beginner-friendly.

## FEATURES

- User Registration  
- User Login  
- Username Validation  
- Duplicate Username Checking  
- Password Verification  
- File Handling in C++  
- Simple Menu-Driven Interface

## TECHNOLOGIES USED

- C++  
- File Handling  
- Functions

- Loops  
# HOW THE PROJECT WORKS

When the program starts, the user is shown three options:

1. Register a new account  
2. Login with an existing account  
3. Exit the program  

During registration, the user enters a username and password. The program checks whether the username already exists. If the username is available, the user data is stored in a text file.

During login, the system reads the stored file and verifies the username and password entered by the user.
